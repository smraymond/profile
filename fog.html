FOG Image Server

FOG, or Free Open-Source Ghosting, is a networking server that clones a machines hard drive, and can then deploy that clone to a new computer. For this project, I created a FOG server to clone a Windows Virtual Machine.

FOG GUI

The FOG GUI is where you can determine which machines will be deployed (imaged) or which will be uploaded (cloned). When a machine is added to the server, a task to deploy or clone can be made.

Creating Hosts and Imaging

When an image needs to be deployed, network boot the machine to the server and add the host to the server. Then click quick image, which selects an image file to load from, to begin the cloning process.
