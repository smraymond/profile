CSUSB Scheduler

View Project Here

The CSUSB Scheduler is a web application I built, with a small group of programmers for my software engineering class, which later became the topic of my senior project.

Creating Schedules

The scheduler uses Google SignIn to allow students and professors of CSUSB to create event reminders for their classes, including homework, quizzes, exams, and more.

Class Organization

Professors can create classes, with class specific events, that students can then enroll into, and view. Each class, also has a discussion board, where students and professors can create and reply to posts.
