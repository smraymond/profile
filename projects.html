Nodejs Project

FreeNAS Project

Documentation for creating a FreeNAS server in a virtual Linux environment, then linking it to an Active Directory account in Windows Server 2012, in order to access those users and files in the NAS.

Documentation for creating a FOG, or Free Open Ghosting, server in a virtual Linux environment in order to demonstrate how to capture and deploy images of different operating systems.
